## Custom dynamic wallpappers for MacOS

Created with [this great cli tool by mczachurski](https://github.com/mczachurski/wallpapper).

Just copy `Weed.heic` to your wallpappers folder and choose it as a regular wallpapper.
___

Also you can use thus `wallpapper.json` as a template. 
Replace my 16 pictures with you own and use [this tool](https://github.com/mczachurski/wallpapper).
